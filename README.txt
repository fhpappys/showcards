Deploy (drag & drop)
--------------------
1) Log into https://app.netlify.com → New Site → Deploy manually.
2) Drag the ENTIRE 'netlify_volunteer_bundle_v4' folder.
3) Visit /volunteer.html and submit a unique test email.
4) Netlify → Forms → 'volunteer' → check All and Spam tabs.
5) Enable email notifications for the form.
